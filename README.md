## Запуск приложения

### Установите зависимости:

### `pnpm install`

### Запустите приложение вместе с сервером JSON Server с моковыми данными для авторизации и контактов:


### `pnpm run start`

Приложение будет доступно по адресу http://localhost:3000.

### Версия Node.js

### `node -v : 18.17.0`
